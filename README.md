# 🎰 MK BINGO DELUXE - Telegram Bot

A full-featured multi-player Bingo game bot for Telegram with Telebirr payment integration, real-time gameplay, and beautiful Web App interface.

![Version](https://img.shields.io/badge/version-3.0.0-blue)
![Node](https://img.shields.io/badge/node-18.x-green)
![License](https://img.shields.io/badge/license-MIT-orange)

## ✨ Features

### 🎮 Game Features
- **Multi-player support** - Play with friends and family
- **Real-time gameplay** - Live number calling with Socket.io
- **Beautiful Web App** - Rich graphical interface inside Telegram
- **Multiple game types** - Full House, 1 Row, 1 Column, 4 Corners, X Shape, Random
- **Automatic winner detection** - No manual checking needed
- **Instant prize distribution** - Winners get paid automatically

### 💰 Payment Integration
- **Telebirr payment** - Ethiopian mobile money support
- **Balance system** - In-game wallet for easy play
- **Transaction history** - Track all payments
- **Auto top-up** - Buy cards directly with Telebirr

### 👑 Admin Features
- **Game control panel** - Start rounds, draw numbers
- **Statistics dashboard** - Track players, cards, revenue
- **Game type selection** - Change winning patterns
- **Prize management** - Set and adjust prizes

### 📊 Technical Features
- **SQLite database** - Lightweight and reliable
- **Docker support** - Easy deployment anywhere
- **Railway ready** - One-click cloud deployment
- **PM2 support** - Production process management
- **Health checks** - Automatic monitoring
- **Logging** - Comprehensive error tracking
